# Popup Modal Box - Confirmation Dialog

Uma caixa de diálogo modal popup moderna, minimalista e totalmente responsiva desenvolvida com HTML5 e CSS3. O projeto simula uma interface de confirmação crítica de sistema (como a exclusão permanente de um arquivo), focando em uma hierarquia visual clara e elementos de ação bem definidos.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do contêiner modal, textos e botões de ação.
* CSS3: Estilização de bordas, alinhamentos, tipografia e gerenciamento de sombras projetadas.
* Flexbox: Centralização perfeita do card sobre a camada de fundo e organização dos botões em linha.

## Funcionalidades e Técnicas Aplicadas

* Overlay de Fundo (Camada Escura): Uso de um plano de fundo cinza fosco para isolar visualmente a janela de confirmação e manter o foco do usuário no aviso.
* Ícone de Alerta Vetorial: Inclusão de um círculo vermelho com um ponto de exclamação centralizado para indicar uma ação de alta prioridade ou irreversível.
* Tipografia Equilibrada: Destaque estrutural no título "Confirm" acompanhado de uma pergunta direta e legível sobre a intenção do usuário.
* Botões de Ação Contrastantes: 
  * Botão de confirmação ("Yes, Delete!") estilizado em azul para guiar a escolha primária.
  * Botão de cancelamento ("Cancel") destacado em vermelho vibrante para alertar o usuário sobre a interrupção da ação.
* Design Limpo e Flutuante: Cantos levemente suavizados com `border-radius` e uma leve sombra (`box-shadow`) que confere profundidade ao card branco.
